[조작키]
Q : 프로그램 종료
1, 3 : 카메라 변경
W, A, S, D : 이동
Space_bar : 점프
마우스 좌클릭 드래그 : 객체 회전, 1인칭 카메라 회전

[, ] : 카메라 공전(3인칭 카메라)
8, 4, 5, 6 : 카메라 이동(3인칭 카메라, 키보드 오른쪽 번호판 텐키 사용 권장)
7, 9 : 카메라 자전(3인칭 카메라)

V : 기둥 등장
B : 미로 생성
N : 낮은 높이 애니메이션
M : 위아래 애니메이션
+, - : 애니메이션 속도 조절

I : 폴리곤 모드(면, 선)
O : 깊이 테스트
P : 투영(원근, 평행)
L : 초기화 후 미로 재생성
